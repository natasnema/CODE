除了常見的boolean/ number/ string 外 JS中還有兩種複合屬性 array 陣列 以及 object 物件 通常而言一個變數只能存單筆資料 array 陣列 與 object 物件 可以存多種不同資料 array 陣列 使用 [X,Y,Z] 表示 陣列中抓取特定資料 使用索引值 (從0開始) 陣列中可以存放不同種類資料 也再放入一個陣列 object 物件 使用 {} 表示 物件以 key 與 value 紀錄 (key 盡量不要用Number) 物件中抓取特定資料 使用" . " (dot), DOM就是將HTML元素模擬成物件讓使用者操作 物件內部可以再放入物件或陣列
